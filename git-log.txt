    Add commit-generation and diagnostic scripts
    
    Introduce four utility scripts: generate_commits.js creates a synthetic commit history (multiple authors, configurable date range, adds/deletes, writes seed files to backend/seedData and commits with GIT_AUTHOR_DATE/GIT_COMMITTER_DATE over 7 commits per author); diag.js runs git add and prints staged --numstat for diagnostics; diag2.js executes an ultimate_rebuild script and reports failures; measure.js writes a .gitignore, stages/resets files, makes a 'measure' commit and prints the latest commit --stat and --numstat. These tools are intended to generate and inspect a controlled repository history for testing/measurement.
